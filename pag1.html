Conversor de Moedas

O projeto de Conversor de Moedas permite a conversão entre diferentes moedas de forma rápida e simples. Ele foi desenvolvido utilizando HTML, CSS e JavaScript puro, sem o uso de APIs externas. O usuário pode selecionar a moeda de origem e destino, e o sistema calculará o valor convertido com base em uma taxa fixa predefinida.
