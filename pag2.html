Calculadora de IMC

O projeto de Calculadora de IMC permite calcular o Índice de Massa Corporal (IMC) com base na altura e no peso do usuário. Ele foi desenvolvido utilizando HTML, CSS e JavaScript. O sistema exibe diferentes mensagens personalizadas dependendo do resultado do IMC, indicando se o usuário está abaixo do peso, com peso normal, sobrepeso ou obesidade.
