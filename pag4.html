Banco Xiiii

O Banco Xiiii é um sistema bancário simples, desenvolvido com foco em funcionalidades como depósito, saque e login de usuários. O projeto foi criado sem o uso significativo de IA, sendo desenvolvido com base em pesquisas, e conta com páginas HTML como index.html, login.html, depositar.html e sacar.html, além de um arquivo script.js para a lógica do sistema e style.css para o design.
